HTML (HyperText Markup Language) — это язык разметки, с помощью которого создаются веб-страницы. Он определяет структуру и содержание страницы: где будет текст, изображение, ссылка, таблица или видео. HTML не является языком программирования — он не выполняет действий, а описывает, как информация должна быть представлена в браузере. Основой HTML являются теги — специальные элементы в угловых скобках, например .В сочетании с CSS (оформление) и JavaScript (динамика) HTML формирует основу любого современного сайта.
