Язык Год создания Краткое описание
HTML 1991 Язык разметки, который используется для создания структуры веб-страниц: текста, изображений, ссылок и других элементов.
CSS 1996 Язык таблиц стилей, определяющий внешний вид веб-страниц — цвета, шрифты, расположение элементов и адаптацию под разные устройства.
JavaScript 1995 Язык программирования, добавляющий интерактивность сайтам — анимации, обработку действий пользователя, динамическое изменение содержимого.
